Define uma data/hora limite para cada turma (campo Section do CSV). Quando configurada, alunos não-graduados dessa turma recebem um aviso adicional no e-mail de cobrança informando o prazo final.

Contas de teste do Canvas (como "Testar aluno") são detectadas e ignoradas automaticamente. Use o botão 🚫 na linha de cada aluno para ignorá-lo manualmente. Aqui você pode revertê-los.

Salva em um único arquivo JSON tudo que você configurou: critérios, assunto do e-mail, datas de encerramento e lista de alunos ignorados.

💡 Dica para cursos longos: as configurações ficam no seu navegador (localStorage) e somem se você limpar dados do navegador, trocar de máquina ou usar modo anônimo. Para cursos de meses, exporte um backup periodicamente e re-importe se precisar.

❓ Como usar

1️⃣ Exportar do Canvas

No Canvas LMS, vá em Notas → Exportar → CSV. Não edite o arquivo no Excel.

2️⃣ Carregar o arquivo

Arraste o CSV para a área indicada ou clique para selecioná-lo. Você verá um preview antes de processar.

3️⃣ Analisar e enviar e-mails

A tabela mostra cada aluno e seu status. Clique em uma linha para ver pendências. Use os filtros coloridos para separar por status. Para enviar e-mails, use as ações em massa ou os botões individuais por aluno.

⌨️ Atalhos de teclado

  • / — Focar na busca
  • Esc — Fechar modais e limpar busca
  • D — Alternar tema escuro

📋 Critérios de status

🟢 OK KC ≥ 70% e Lab ≥ 95%
🔴 Crítico KC < 70% e Lab < 95%
🟡 Atenção Apenas um dos critérios atingido
🎓 GraduadoColuna Graduated Final Points = 1

Os critérios são editáveis em ⚙️ Configurações.
